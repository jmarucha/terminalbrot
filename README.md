# Terminalbrot

An interactive Mandelbrot set explorer for the terminal, written in Haskell to see if functional programming is actually fun

## Features

- It's interactive!

Seriously, how many Haskell programs can do that?

## Building

Use cabal

```bash
cabal build
```

## Running

Use cabal

```bash
cabal run
```

## Controls

- `W` or `↑` - Move up
- `S` or `↓` - Move down
- `A` or `←` - Move left
- `D` or `→` - Move right
- `Q` - Zoom in
- `E` - Zoom out
- `ESC` - Exit

## How It Works

Via escape time colouring algorithm. Nothing fancy. Randomizing characters based on escape time give some nice details.
